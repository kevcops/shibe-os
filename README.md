# 🐕 Shibe OS

Welcome to **Shibe OS** — your secure, professional dental office portal built on Fedora's next-generation immutable technology!

---

## 🛡 About

**Shibe OS** is a customized, secure, and professionally branded Linux operating system for dental offices.

✅ Immutable Read-Only System  
✅ KDE Plasma Desktop Environment  
✅ Dental-Specific Setup (CareStack Integration)  
✅ Optional App Installations via First Boot GUI  
✅ Automatic Nightly System Updates  
✅ Lightweight, Fast, and Highly Secure  

---

## 🚀 Quick Start

1. Rebase to Shibe OS (on Fedora Silverblue or Kinoite):

    ```bash
    rpm-ostree rebase ostree-unverified-registry:ghcr.io/kevcops/shibe-os:latest
    ```

2. Reboot:

    ```bash
    systemctl reboot
    ```

3. Follow Onboarding:
    - Enter CareStack Office ID
    - Install optional apps if needed

---

## 🛠 Maintenance

- **System Updates**: automatic nightly bootc upgrades
- **Manual Upgrade** (optional):

    ```bash
    sudo bootc upgrade
    sudo systemctl reboot
    ```
